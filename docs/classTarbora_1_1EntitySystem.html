Detailed Description

The system that manages the basic info about actors.

It owns three components: Tarbora::TransformComponent, Tarbora::InfoComponent and Tarbpora::TypeComponent.

Subscribed to:

set_position (Message::ApplyPhysics). Sets the actor to a new position.

set_orientation (Message::ApplyPhysics). Sets the actor to a new orientation.

move (Message::ApplyPhysics). Increments the actor's current position.

rotate (Message::ApplyPhysics). Increments the actor's current orientation.

Triggers:

move_actor (Message::MoveActor) every frame for every actor whose transform changed during that frame.
